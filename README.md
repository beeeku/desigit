# desigit
Git command alternative for Desi People

# Installation
  1. clone repo
  2. cd into directory
  3. npm install 
  4. npm link
  



# Usage: desigit [options] [command]


  Commands:

    ki_haal
    kheench
    pell [alias] [branch]
    laa [bookmark]
    dikha [commit]
    pichla
    nakal [url]
    aarambh [projectName]
    milade [branch]
    ghur [branch]
    shakha [options] [branchName]
    wapis [options] [commit]
    pakka "[message]"
    daalo [file]
    antarr [options]

  Options:

    -h, --help     output usage information
    -V, --version  output the version number
