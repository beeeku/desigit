# desigit
Git command alternative for Desi People

# Installation
  cd Directory /n 
  npm install desigit
  npm install /n
  npm link /n
  



  Usage: desigit [options] [command]


  Commands:

    ki_haal
    kheench
    pell [alias] [branch]
    laa [bookmark]
    dikha [commit]
    pichla
    nakal [url]
    aarambh [projectName]
    milade [branch]
    ghur [branch]
    shakha [options] [branchName]
    wapis [options] [commit]
    pakka "[message]"
    daalo [file]
    antarr [options]

  Options:

    -h, --help     output usage information
    -V, --version  output the version number
